# Lily58 with Luna

Custom layout for Lily58 with dual encoders and dual OLED.

Currently running [Luna](https://github.com/HellSingCoder/qmk_firmware/tree/master/keyboards/sofle/keymaps/helltm) and some weird custom logos.
